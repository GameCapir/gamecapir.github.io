About

Capir is a puzzle game where you rotate groups of pieces to match a solution.

To rotate the pieces, press on the first one and release on the last in line.


Capir has two game modes: challenge mode and free mode.

Challenge mode

In challenge mode each puzzle is custom made, the moves to solve the puzzle have been carefully selected.

In that mode the puzzles can be solved in 3, 4, or 5 moves, depending on the difficulty level.

3 4 5

Pieces in challenge mode have dots that hint how many times they should be rotated.

Free mode

In free mode each puzzle is randomly generated and the amount of moves to solve them is not decided.

The best scores in free mode are recorded in a global leaderboard, you compete against the world!

Game screen

In challenge mode

  • a: Current moves / max moves
  • b: Current level
  • c: Show the puzzle solution
  • d: The shape to match

In free mode

  • a: Current moves / max moves
  • b: Current level
  • d: The shape to match

Navigation

  • e: Go to start screen
  • f: Go to setup screen
  • g: Next puzzle
  • h: Previous puzzle
  • i: Restart level

Setup screen

In challenge mode

  • a: Player moves on the selected level
  • b: Selected level identification
  • c: Selected level
  • d: Complete level
  • e: Incomplete level
  • f: Locked level
  • g: Go to game screen
  • h: Go to progress screen
